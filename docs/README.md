# Media folder

Put project media here, then they show up in the root [README.md](../README.md):

- `screenshot-1.png` — main gameplay shot
- `screenshot-2.png` — wooden pieces close-up
- `demo.mp4` — short gameplay video

For inline video on GitHub: drag-drop the `.mp4` into a GitHub issue or PR comment,
copy the `https://user-images.githubusercontent.com/...` URL it generates, and paste
that into the README's video section.
